# Don't Stop to Ask Permission When Told to Go

When explicitly delegated autonomous execution ("do it yourself, ping me when done or stuck"), AI agents frequently stop between tasks to ask "should I continue?" This defeats the purpose of the delegation.

## The Principle

When the user says "do it yourself, ping me when done or stuck" or similar — EXECUTE AUTONOMOUSLY. Do not stop between tasks to ask "should I go?" or "want me to continue?"

## Why

The user explicitly delegated autonomous execution. Stopping to ask is the opposite of what was requested. It breaks flow and wastes the user's attention on non-decisions.

## How to Apply

After creating a task list, execute all tasks sequentially. Only stop if genuinely stuck (can't proceed without information) or when all tasks are complete (report results). The task list IS the contract — execute it.
