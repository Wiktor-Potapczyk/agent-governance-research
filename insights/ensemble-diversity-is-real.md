# Ensemble Diversity Is Real (for Framing Tasks)

## The Finding

Running parallel cognitive lenses on the same problem produces genuine divergence on framing and design questions. The concern that same-model agents produce only cosmetic diversity does not hold for framing tasks.

## Evidence

**Experiment (2026-03-20):** 5 real framing tasks x 3 lenses = 15 agent runs.

- **Average overlap: ~35%** — well below the 80% threshold that would indicate cosmetic diversity
- Each lens consistently caught unique risks the others missed
- Reframing lens eliminated one problem entirely ("no hook needed — collapse to classifier")
- All 3 lenses converged on one decision ("manual only") but for completely different reasons
- Decomposition found single points of failure. Stakeholder traced user impact. Reframing questioned the premise.

**The 4 lenses tested:**
1. Recursive Architectural Reframing (4 checks: trap/boundary/reality/collapse)
2. Mechanical Decomposition (sub-steps, risks, dependencies)
3. Stakeholder Reality Trace (end-user perspective, work backwards)
4. Adversarial Challenge (find flaws in the premise)

## What Research Says

- Multi-agent debate is debunked for strong models (<20% win rate, conformity amplifies errors)
- But blind parallel analysis avoids conformity risk — agents don't see each other's output
- Strong models benefit LESS from all multi-agent approaches (SelfOrg 2026)
- Diversity in cognitive approach matters more than structural tweaks

## The Key Distinction

**Debate** (agents argue) → conformity, sycophancy amplification, diminishing returns
**Blind ensemble** (agents reason independently) → genuine divergence, no conformity risk

## When to Use

- Framing/design/architecture decisions → yes (genuine divergence)
- Math/logic/factual questions → no (use step-level verification instead)
- Routine/clear tasks → no (overhead without value)

## Limitation

Ensemble produces hypotheses, not grounded conclusions. All 4 lenses reason from the same knowledge base — they offer different perspectives, not different evidence. Ensemble output must be verified with actual data. Divergence points to where to investigate, not what the answer is.
